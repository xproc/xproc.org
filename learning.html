XProc - Learning

We have collected as many learning materials as possible about XProc. This is all about version 3.0 (and also usable for 3.1).

Book

The XProc 3.0 Programmer Reference by Erik Siegel is available for sale here.

XProcRef is an accompanying support website with user/programmer oriented reference information about the built-in steps.
