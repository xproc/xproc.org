XProc 3.0 - Processors

There are two XProc 3.0 processors, currently (September 2022) under development. Both also have an XProc 1.0 version available.

MorganaXProc-III

The MorganaXProc-IIIse processor is available. It implements all required features of the core language as well as all steps in the standard step library. Additionally most optional features are supported as well as the steps in the File Steps Library and the Validation Steps Library.

XML Calabash

The XML Calabash processor is being developed.
