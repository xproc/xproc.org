XProc 3.0 - Processors

There are two XProc 3.0 processors, currently (June 2020) still under development. Both also have an XProc 1.0 version available.

MorganaXProc-III

The MorganaXProc-III processor is available as a beta release that implements most of the specification.

XML Calabash

The XML Calabash processor is being developed.
